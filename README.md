Axiom - An AWS focused config manager

## Packages

- `@dsmrt/axiom-cli`
- `@dsmrt/axiom-aws-sdk`
- `@dsmrt/axiom-config`

## Features
- Multiple environments/accounts/regions
- Secrets with SSM Parameters

## Getting Started

### Install the CLI

```bash
npm install axiom-cli
```

### CLI Usage

#### Features
- View rendered configs
- Manage SSM Parameters

```
axiom --help
```
