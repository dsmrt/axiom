# Axiom - CLI

> An AWS focused config manager

## Features
- Multiple environments/accounts/regions
- Secrets with SSM Parameters

## Getting Started

### Install the CLI

```bash
npm install @dsmrt/axiom-cli
```

### CLI Usage

#### Features
- View rendered configs
- Manage SSM Parameters

```
axiom --help
```
